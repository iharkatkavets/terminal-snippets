## Windows
<br>`Ctrl+w,s` - Split window horizontally
<br>`Ctrl+w,v` - Split window vertically
<br>`Ctrl+w,q` - Close the window
<br>`Ctrl+w,Ctrl+w` - Switch between windows
<br>`Ctrl+w,[i,j,h,l]` - Navigate to adjacent window
