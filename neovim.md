Windows
Split window horizontally
```
Ctrl+w,s
```
Split window vertically
```
Ctrl+w,v
```

```
Ctrl+w,q
```

```
Ctrl+w,Ctrl+w
```
Navigate to adjacent window
```
Ctrl+w,[i,j,h,l]
```
Swap the order down
```
Ctrl+W,J
```
Swap the order up
```
Ctrl+W,K
```
